sort() Method

sort elements in an array The sort() method sorts the elements of an array in place and returns the reference to the same array, now sorted. The default sort order is ascending, built upon converting the elements into strings, then comparing their sequences of UTF-16 code units values. The sort() method allows you to sort elements of an array in place. Besides returning the sorted array, the sort() method changes the positions of the elements in the original array. By default, the sort() method sorts the array elements in ascending order with the smallest value first and largest value last.

Syntex: // Functionless sort() // Arrow function sort((a, b) => { /* … */ } ) // Compare function sort(compareFn) // Inline compare function sort(function compareFn(a, b) { /* … */ })

The sort() method casts elements to strings and compares the strings to determine the orders. Consider the following example: let numbers = [0, 1 , 2, 3, 10, 20, 30 ]; numbers.sort(); console.log(numbers); Output:[ 0, 1, 10, 2, 20, 3, 30 ] In this example, the sort() method places 10 before 2 because the string “10” comes before “2” when doing a string comparison. To fix this, you need to pass a compare function to the sort() method. The sort() method will use the compare function to determine the orders of elements. let numbers = [0, 1 , 2, 3, 10, 20, 30 ]; numbers.sort( function( a , b){ if(a > b) return 1; if(a < b) return -1; return 0; }); console.log(numbers); //[ 0, 1, 2, 3, 10, 20, 30 ] //With Arrow Function let numbers = [0, 1, 2, 3, 10, 20, 30]; numbers.sort((a, b) => a - b); console.log(numbers); //[ 0, 1, 2, 3, 10, 20, 30 ]

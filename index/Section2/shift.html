Shift() Method

Remove the first element from an array. The Array.prototype.shift() method removes the first element from an array and returns that element. The following shows the syntax of the shift() method: Syntex: array.shift() If the array is empty, the shift() method returns undefined. Otherwise, it returns the removed element. Also, the shift() method reduces the length property of the array by one. Use the shift() method with an array-like object via the call() or apply() method.
